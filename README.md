# Cryptography Puzzle Hunt System

This is a C++ based educational game that improves logical thinking and understanding of cryptographic concepts through interactive puzzles such as Caesar cipher, Affine cipher, modular arithmetic, and prime number challenges.

## Installation Instructions

1. Download or clone the repository.
2. Open the project folder.
3. Compile the program:

   g++ src/main.cpp -o puzzle

4. Run:

   puzzle

## Usage Examples

User enters roll number → selects difficulty level → solves puzzles → score is calculated and stored.

Example:
Encrypted: khoor  
Input: hello  
Result: Correct Answer

## Dependencies

- C++ Compiler (GCC / MinGW / MSVC)
